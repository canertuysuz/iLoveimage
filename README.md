# iLoveimage

iloveimage is a tool to download photos from web pages in bulk

## Installation

After downloading git clone or zip, download requirements

```bash
pip3 install -r requirements.txt
```
And run this code
```bash
python main.py --help
```

## Usage

```python
main.py [-h] [-p PATH] <url>
```
